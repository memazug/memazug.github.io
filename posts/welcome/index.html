Welcome to the Memphis Azure User Group

Posted on 17 Jul 2023

Welcome to the Memphis Azure User Group!

Microsoft Azure is a leader in cloud computing services. Technology professionals who want to succeed in the industry should add Azure to their toolkit. The Memphis Azure User Group is a resource for those professionals to learn about Microsoft Azure, network with other technology professionals, and strengthen the Memphis technology community.

Meetings will be posted on memphisazure.com, on Meetup and promoted on Twitter at @MemphisAzure. Please use Meetup to RSVP for the meetings.

If you are interested in speaking at or sponsoring the Memphis Azure User Group, or want to suggest a topic or speaker, please contact douglas@memphisazure.com.

The Memphis Azure User Group maintains a Code of Conduct which can be found here.
