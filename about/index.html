About The Memphis Azure User Group

Posted on 11 Jul 2023

The Memphis Azure User Group meets virtually, via Microsoft Teams, on the first Monday of every month at 6:00PM Central time.

All participants are expected to follow the Code of Conduct. Please review it.

Follow the Memphis Azure User Group on Twitter at @MemphisAzure.

Recorded meetings will be posted to YouTube.

If you are interested in speaking to or sponsor the group, or would like to suggest a speaker or topic, please contact douglas@memphisazure.com.
